# Cozy Paws

Оригинальная локальная браузерная игра про виртуальных питомцев. Проект вдохновлен жанром мобильных pet-games, но не использует чужие бренды, изображения, тексты, CSS, HTML или уникальные элементы дизайна.

## Возможности

- регистрация, вход и выход;
- защищенные игровые страницы;
- профиль игрока и игровая валюта;
- стартовый питомец после регистрации;
- создание и выбор питомцев;
- карточка питомца с уровнем, опытом, энергией, настроением и сытостью;
- действия: покормить, поиграть, погладить, отправить гулять;
- магазин предметов и инвентарь;
- использование предметов из инвентаря;
- ежедневные задания с наградами;
- рейтинг игроков;
- Django admin и seed-команда для стартовых данных;
- адаптивная мобильная верстка с локальными CSS/JS.

## Быстрый запуск

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
python manage.py migrate
python manage.py seed_game
python manage.py runserver
```

Открой в браузере: http://127.0.0.1:8000/

После `seed_game` доступен тестовый пользователь:

```text
login: demo
password: demo12345
```

## Админ-панель

Создай администратора:

```powershell
python manage.py createsuperuser
```

Затем открой http://127.0.0.1:8000/admin/
